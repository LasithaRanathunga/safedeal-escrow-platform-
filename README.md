SafeDeal is a web application designed to enable secure and transparent freelance collaborations. It provides a structured workflow for managing contracts, tracking milestones, and handling file exchanges between clients and freelancers.

Freelance work often lacks trust—clients may not pay after receiving work, and freelancers may disappear after getting paid. SafeDeal addresses this through an escrow-based workflow where freelancers provide a preview of their work, while the final version is only accessible after payment, protecting both parties. Work can also be divided into smaller milestones, allowing payments to be released incrementally as progress is verified. This reduces risk, improves transparency, and keeps both sides aligned throughout the project.

## CI/CD Pipeline 

Implemented a CI/CD pipeline that automatically builds and deploys each pull request to a dedicated preview environment. 

- 🌐 Generates a unique preview URL for every PR
- 🔗 Links the environment directly to the pull request
- 🔍 Enables developers and testers to validate changes instantly
- 🧪 Provides an isolated, production like environment per PR
